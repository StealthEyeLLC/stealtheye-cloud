# StealthEye Skill — Tab Resume

## Purpose

Resume a StealthEye Cloud mission from Relay, Seal, Active, Decisions, and Next Action artifacts.

## Procedure

1. Read `STEALTHEYE_RELAY.md`.
2. Read `STEALTHEYE_RELAY.json`.
3. Read `STEALTHEYE_SEAL.json`.
4. Read `STEALTHEYE_ACTIVE.md`.
5. Read `STEALTHEYE_DECISIONS.md`.
6. Perform `NEXT_ACTION.md` unless a boundary is required.

## Rules

1. Do not ask the user to re-explain.
2. Do not reopen frozen decisions.
3. Treat the latest Seal as checkpoint truth.
4. Stop only for true boundaries.

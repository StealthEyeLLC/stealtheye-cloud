# Release Candidate

## Purpose

This document defines the first public-safe StealthEye Cloud release-candidate posture.

## S5 release candidate scope

S5 is a public proof candidate, not a commercial release.

It requires:

1. no local/laptop requirement
2. no secrets
3. no paid compute requirement
4. green Linux fast proof
5. green Linux full proof
6. green browser proof
7. green targeted Windows proof when triggered
8. complete schema inventory
9. valid Relay and Seal
10. public-safe proof canvas

## Required release artifacts

1. `docs/S5_FINAL_REPORT.md`
2. `docs/RELEASE_CANDIDATE.md`
3. `docs/FIRST_E2E_MISSION.md`
4. `public/proof/index.html`

## Boundary

Release candidate status does not mean marketplace launch, paid service launch, or customer deployment. It means the public repository can prove the current no-local ChatGPT-native agent spine through GitHub Actions.

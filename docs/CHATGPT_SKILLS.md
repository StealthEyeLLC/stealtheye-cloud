## Rules

1. Skills are instructions and templates, not hidden authority.
2. Skills must obey Active, Decisions, Relay, Seal, and Next Action.
3. Skills must not claim background powers ChatGPT does not have.
4. Skills must preserve public/private boundaries.
5. Skills must stop for true boundaries.

# Background Capability Reality

## Purpose

This document prevents hidden-autonomy claims and keeps StealthEye Cloud aligned with real ChatGPT/GitHub surfaces.

## What exists

StealthEye Cloud can use:

1. the active ChatGPT tab
2. ChatGPT Skills
3. ChatGPT Tasks as scheduled nudges
4. Codex when user-launched or configured
5. GitHub Actions as the no-local proof body
6. Deep Research when user-started
7. Agent Mode when user-started
8. Apps/connectors where available
9. Relay and Seal for handoff continuity

## What must not be claimed

StealthEye Cloud must not claim ChatGPT can secretly create tabs, silently launch Deep Research, silently launch Agent Mode, or spend paid compute without approval.

## S3 status

S3 represents real worker and background surfaces through public-safe docs, schemas, Skills, and registry crates. It does not create hidden background execution.

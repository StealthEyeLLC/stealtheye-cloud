# StealthEye Cloud S1 Drop Package

This branch contains a GitHub-hosted copy of the S1 drop package.

The package file is stored as Base64 at:

`drops/s1/StealthEye_Cloud_S1_Mission_Executor_Drop.zip.b64`

Decode it to a ZIP, expand it outside the repo, run the included PowerShell apply script against `X:\stealtheye-cloud`, then commit and push branch `build/s1-mission-executor-atomic-drop`.
